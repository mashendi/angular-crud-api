# Angular-CRUD-API

Simple Angular SPA CRUD system API using json-server

## Perquisites

- Angular CLI

  ```bash
  npm i -g @angular/cli
  ```

- json-server
  ```bash
  npm i -g json-server
  ```

## Installation

- Install npm packages

  ```bash
  npm i
  ```

- run json-server

  ```bash
  json-server --watch db.json
  ```

- run app
  ```bash
  ng serve -o
  ```
